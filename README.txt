Title: Appointment Scheduler
Purpose: allow different users to manage customers and schedule appointments at a specific time with customers using database to store records of customers and appointments
Author: Rui Huang
Contact info:rhuang3@umn.edu
Application version: 1.0
Date: 5/7/2022
IDE:IntelliJ IDEA 2021.3.3 (Community Edition)
JDK: jdk-18 JAVA_VERSION="18" JAVA_VERSION_DATE="2022-03-22"
JavaFX: Version: 18
How to run the program:
user command prompt or any Java IDE to run main.class file
after the program is initiated, log in to the main form with correct user name and password
user to main form to add/modify/delete customers or appointments
use the report 1/2/3 button to access 3 different reports
Report 3:
user can choose to display appointments create by different users
use the combo box to choose users and the tableview will display the appointments created by the chosen user
 mySql connector: mysql-connector-java-8.0.26
